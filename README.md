# 🎵 Java Console Audio Player

A simple Java console-based audio player that uses the `Clip` API to play sound files.

This project allows users to control audio playback directly from the terminal.

---

## 🚀 Features

* Play audio ▶️
* Stop audio ⏹️
* Reset audio 🔄
* Quit program ❌
* Basic error handling for audio issues

---

## 🛠️ Technologies

* Java
* `javax.sound.sampled`
* Scanner (user input)

---

## ▶️ How to use

1. Clone the repository
2. Add your own `.wav` file to the project folder
3. Update the file path in the code:

```java
String filePath = "audio.wav";
```

4. Run the program

---

## ⚠️ Note

The audio file is not included due to size limitations.

You can use any `.wav`, `.au`, or `.aiff` file, but make sure the path is correct.
